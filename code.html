The Code for TacoCat


                


                

                    
A brief summary

                    
The user enters a string value into the text box.
                        After pressing the start button the JS grabs the value from the text box.
                        That value is turned into an array and reversed. Then the reversed array is converted
                        back into a string. The function checks if the orignal string and the reveresed string 
                        are equal. It returns the result to the html element and makes it visible.
                        The Hide() function hides the result textbox after a user clicks on the textfield.
